Entities and Attributes

Profile

Users will be able to create a profile that allows them to like and upload new murals.

One to one - one user can create one profile

  • profileId (primary key)
  • profileActivationToken
  • profileEmail
  • profileHash
  • profileName
  • profileUsername

Mural

Murals will be submitted by users as well as by us developers.

Many to Many - many murals can be submitted by many users

  • muralId (primary key)
  • muralLikeProfileId (foreign key)
  • muralProfileId (foreign key)
  • muralContent
  • muralDate
  • muralImage

Main Map

This is the entity for the large overview map with its page.

One to Many - one main map will have many drop points

  • mainMapId (primary key)
  • mainMapDropPointId (foreign key)
  • mainMapMuralId (foreign key)
  • mainMapContent

Route Maps

This is the entity for the routes page where pre-mapped routes are available based on walk time.

Many to Many - there will be many routes with many drop points

  • routeId (primary key)
  • routeDropPointId (foreign key)
  • routeMuralId (foreign key)
  • routeMapContent

Drop Points

This is the entity for the points on the map indicating the location of a mural.

One to Many - One drop point per mural, with many drop points per map.

  • dropPointId (primary key)
  • dropPointMuralId (foreign key)
  • dropPointProfileId (foreign key)
  • dropPointLocation

Like

This is the entity for the likes, which will be available on every mural.

Many to Many - There can be many likes on many murals

  • likeProfileID (foreign key)
  • likeMuralId (foreign key)
